# Competitor logos for the /compare page

The comparison table renders official competitor logos IF a file exists here,
otherwise it falls back to a clean text wordmark of the platform name.

To use real logos, download the OFFICIAL brand asset from each company's
press/brand resources and save it here as an SVG named by platform id:

- shopify.svg       → https://www.shopify.com/brand-assets
- squarespace.svg   → https://www.squarespace.com/press/downloads (brand kit)

Notes
- Use the official asset only. Do not recreate or redraw trademarked logos.
- Comparison pages that name competitors and use their logos rely on
  "nominative fair use" — factual, non-misleading comparison. Keep all claims
  accurate and do not imply endorsement. If in doubt, the text wordmark
  fallback is always safe.
- SVGs render best. A monochrome / single-colour version usually sits most
  cleanly next to the Evercart mark.
